# Excel-Travel-Booking-Analytics

## 📌 Project Overview

This project focuses on analyzing travel booking data using Microsoft Excel to derive actionable business insights. The goal was to clean raw data, perform analysis, and build an interactive dashboard to support decision-making for a travel booking company.

---

## 🎯 Objectives

* Analyze booking trends and revenue performance
* Identify top destinations and high-performing agents
* Understand customer behavior and booking patterns
* Track cancellations and payment preferences
* Build a dynamic dashboard for business insights

---

## 🛠️ Tools & Skills Used

* **Microsoft Excel**
* **Power Query (Data Cleaning & Transformation)**
* **Pivot Tables & Pivot Charts**
* **Advanced Excel Functions** (IF, SUMIFS, COUNTIFS, TEXT)
* **Data Visualization (Charts, KPI Cards, Slicers, Timeline)**

---

## 📂 Dataset

* 1000+ travel booking records
* Includes fields like:

  * Booking ID, Date, Customer Details
  * Source & Destination
  * Travel Type, Agent, Channel
  * Revenue, Discount, Payment Mode
  * Booking Status, Rating

---

## 🔄 Project Workflow

### 1. Data Import

* Imported CSV dataset into Excel using Power Query 
[Dataset](Data/travel_booking_dataset.csv)

### 2. Data Cleaning (Power Query)

* Corrected data types (Date, Numeric, Text)
* Handled missing values (Return Date, Rating)
* Removed duplicate records
* Standardized categorical values

### 3. Data Transformation

* Created calculated columns:

  * Revenue = (Base Price - Discount) × Passengers
  * Month-Year for trend analysis
  * Customer Segmentation using IF logic

### 4. Data Analysis

* Built Pivot Tables to analyze:

  * Revenue by destination
  * Booking distribution by channel
  * Agent performance
  * Monthly revenue trends
  * Cancellation patterns
  * Payment preferences

### 5. Dashboard Creation

* Designed an interactive Excel dashboard with:

  * KPI Cards (Revenue, Bookings, Cancellation Rate, Avg Rating)
  * Charts (Bar, Line, Pie)
  * Slicers (Channel, Agent, Travel Type)
  * Timeline filter (Date-based analysis)

---

## 📊 Key Business Insights

1. Goa is the highest revenue-generating destination
2. Online channel contributes the majority of bookings
3. SaiRam Travels is the top-performing agent
4. Revenue peaks during mid-year travel season
5. Higher cancellations observed in offline bookings
6. Goa has the highest booking demand
7. Premium packages receive the best customer ratings
8. Age group 25–40 contributes most bookings
9. UPI is the most preferred payment method
10. Package travel generates the highest revenue
